# pyur
An AUR helper that tries to replicate exactly how `pacman` output.

# Usage
1. Installing packages:
```
$ python main.py [packages]
```
2. Uninstalling packages: Please use `pacman -R` itself.
3. Clear cache:
```
$ rm -r ~/.cache/pyur
```
